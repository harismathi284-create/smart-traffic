# Smart Traffic Management System

## Description
A simple web-based Smart Traffic Management System developed using HTML, CSS, JavaScript, PHP, and MySQL.

## Features
- Traffic Dashboard
- Vehicle Count
- Traffic Report
- Signal Status
- Automatic Signal Update

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
